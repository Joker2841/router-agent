Zero-Token Routing Agent

A local-first agent that answers tasks for free and clears the accuracy gate without spending Fireworks tokens.

Hybrid Token-Efficient Routing AgentBuilt with Gemma on AMD
The problem

Enterprises want lower AI cost without losing quality

Track 1 scores agents on how few Fireworks tokens they spend, as long as they stay above an 80 percent accuracy gate. Local inference counts as zero tokens, so the ideal agent answers everything it can on its own and pays for a premium model only when it truly has to.

Our approach

Answer locally, prove the answer, escalate only if forced

  • A small Gemma model runs inside the container on CPU, so its answers cost zero tokens.
  • Lightweight rules classify each task and route it to the right handler.
  • Math is turned into a short Python program that we run and verify, so the answer is computed, not guessed.
  • Generated code is compiled to confirm it works before we return it.
How it works

One pipeline across eight task categories

Verified locally

Math and logic through reasoning and execution. Code generation and debugging with a compile check.

Answered locally

Sentiment, summarization, named entity recognition, and factual questions handled directly by Gemma.

Escalation

A Fireworks call only when no local answer is produced. In practice this is rarely needed.
